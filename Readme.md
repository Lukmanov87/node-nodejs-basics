### Commands:

* __File System__
  * `npm run fs:copy`
  * `npm run fs:create`
  * `npm run fs:delete`
  * `npm run fs:list`
  * `npm run fs:read`
  * `npm run fs:rename`


* __CLI__
  * `npm run cli:args`
  * `npm run cli:env`


* __Hash__
  * `npm run calcHash`

* __Zlib__
  * `npm run zip:compress`
  * `npm run zip:decompress`

* __Streams__
  * `npm run streams:read`
  * `npm run streams:write`
  * `npm run streams:transform`
